5. Groups of selectors


A comma-separated list of selectors represents the union of all elements selected by each of the ind selectors in the list. (A comma is U+002C.) For example, in CSS when several selectors share the s declarations, they may be grouped into a comma-separated list. White space may appear before an comma.



CSS example:


In this example, we condense three rules with identical declarations into one. Thus,


h1 { font-family: sans-serif }

h2 { font-family: sans-serif }

h3 { font-family: sans-serif }


is equivalent to:


h1, h2, h3 { font-family: sans-serif }
